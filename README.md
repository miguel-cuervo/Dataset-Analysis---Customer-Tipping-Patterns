# Dataset-Analysis - Customer Tipping Patterns

* **Descripción:** Este proyecto surge como una manera de poder enfocar conocimientos adquiridos previamente en la aplicación de Excel a un enfoque centrado en el análisis y visualización de datos. Se descargó y analizó un dataset público en Kaggle referente a propinas, se busca identificar factores y características clave que impactan la generosidad de los clientes. El objetivo es proponer recomendaciones efectivas al gerente o encargado del restaurante.
* **Proceso:** Se buscaba seguir los 6 pasos fundamentales en el análisis de datos: Ask (preguntar), Prepare (preparar), Process (procesar), Analyze (analizar) y Act (actuar). Se utilizó la aplicación de Excel para analizar más de 200 cuentas, filtrando y promediando datos al igual que generando sus respectivas visualizaciones.
* **Herramientas Usadas:** Excel (Tablas dinámicas).
